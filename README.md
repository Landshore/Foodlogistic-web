# FoodLogistic Web

Aquesta és la pàgina web corporativa creada com a proposta per a FoodLogistic S.A. El lloc està desenvolupat amb eines modernes i desplegat mitjançant GitHub Pages.

## 🌐 Web publicada
https://landshore.github.io/Foodlogistic-web/

## 🛠 Eines utilitzades
- **HTML5** per a l’estructura del lloc  
- **CSS3** per al disseny i estil visual  
- **JavaScript** per a petites interaccions  
- **Git & GitHub** per al control de versions  
- **GitHub Pages** per al desplegament públic  
- **StatCounter** per al seguiment de visites  

## 📄 Descripció breu
El projecte consisteix en una web corporativa senzilla i funcional que presenta els serveis de FoodLogistic S.A. i serveix com a demostració professional per al client. El lloc és totalment estàtic i optimitzat per a una càrrega ràpida i una navegació clara.
